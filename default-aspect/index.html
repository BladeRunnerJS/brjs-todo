Source

BladeRunnerJS (BRJS) is an open source developer toolkit and lightweight front-end framework for building modular large-scale HTML5 single page apps. It consists of a set of conventions, developer tools and JavaScript micro-libraries that make it easy to develop, test, deploy and maintain complex JavaScript apps. BRJS apps are split into discrete modules called Blades, each of which implements a single high level feature that can be developed in isolation.
